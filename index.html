Narcissistic Number Check

Hint: A Narcissistic Number is a number which is the sum of its own digits, each raised to the power of the number of digits.

For example:

  1. 153 (3 digits): 13 + 53 + 33 = 1 + 125 + 27 = 153
  2. 1634 (4 digits): 14 + 64 + 34 + 44 = 1 + 1296 + 81 + 256 = 1634

Please enter positive integers only.
